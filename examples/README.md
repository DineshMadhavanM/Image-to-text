# Example Images

This directory contains example images that can be used with the Sports Image Captioning AI Agent.

## How to Use

1. Place your sports images in this directory
2. Run the script and provide the path to the image when prompted

## Example Command

```bash
python sports_captioner.py
```

Then enter the path to an image when prompted, for example:
```
> examples/example_cricket.jpg
```

## Note
- Supported formats: JPG, JPEG, PNG
- For best results, use clear images with visible sports action
