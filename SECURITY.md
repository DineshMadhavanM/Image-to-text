# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities by creating an issue in the issue tracker.

- **Type of Issue**: Security Vulnerability
- **Impact**: Describe the potential impact of this vulnerability
- **Steps to Reproduce**: How can we reproduce this issue?
- **Expected behavior**: What should happen?
- **Actual behavior**: What actually happens?

We will review your report and respond as soon as possible. Thank you for helping keep our project secure!
